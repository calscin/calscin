Calscin is a programming language focusing on allowing everyone to build reliable, efficient and safe software. 

Calsin is the controlled rewrite of [Quickfall](https://github.com/Quickfall/quickfall)

Calscin is currently in heavy development and should not be used for development.

Calscin plans on doing the following
- Thread safety enforcment at compile time (thread markers)
- Memory safety enforcment at compile time (borrow checking)
